Authentication

POST /api/login/

  • Description: Authenticate a user and retrieve a token.

Request

POST /api/login/ HTTP/1.1
Host: 127.0.0.1:8000
Content-Length: 49
Content-Type: application/json

{"username":"subscriber","password":"subscriber"}

Response:

HTTP/1.1 200 OK
Content-Type: application/json

{"token":"2cc76ee3a344307c6d3a37a99260adba81359bc2"}
Authorization
  • Description: This apps authorize user to access API via Authorization header.

Request

GET /api/<endpoint> HTTP/1.1
Host: 127.0.0.1:8000
Authorization: Token <token>
